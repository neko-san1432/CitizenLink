# Team Meeting Minutes and Attendance Logs

## Purpose
Centralized place for meeting minutes, decisions, and attendance tracking.

---

## Meeting Minutes Template

- Meeting date: YYYY-MM-DD  
- Meeting time: HH:MM (timezone)  
- Location / call: (e.g., Zoom link)  
- Facilitator:  
- Note taker:

### Attendees
- Present:
  - Name (Role) — notes
- Absent:
  - Name (Role) — reason (if provided)

### Agenda
1. Topic A — owner
2. Topic B — owner
3. Other items

### Minutes / Discussion
- Topic A:
  - Summary of discussion
  - Key decisions
  - Risks / notes
- Topic B:
  - Summary of discussion
  - Key decisions

### Action Items
| Action | Owner | Due date | Status |
|---|---:|---:|---|
| e.g., Draft user survey questions | Alice | 2026-02-01 | Open |

---

## Attendance Log (rolling)
Use this table to track recurring meeting attendance.

| Date | Meeting type | Attendee | Present (Y/N) | Notes |
|---|---|---|---:|---|
| 2026-01-15 | Sprint Planning | Alice | Y | On time |
| 2026-01-15 | Sprint Planning | Bob | N | Out sick |

---

## File / Storage Guidelines
- Save finalized minutes as `docs/meetings/YYYY-MM-DD-<brief-title>.md`
- Keep action items updated in project tracker (link) and reference issue IDs where applicable.
- If privacy-sensitive info appears, redact before committing.
